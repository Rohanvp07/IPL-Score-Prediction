# IPL-Score-Prediction
This is a machine learning model made using python and flask which predicts first inning score of an IPL match.

# Dependencies
* Python - 3.7
* Scikit-Learn
* Pandas
* Numpy
* Matplotlib
* Seaborn

# Algorithms Used:
* Linear Regression
* Ridge Regression
* Lasso Regression
* Random Forest Regression
